COOKBOOK

Book Design

A book that contains a wide variety of authentic Italian recipes, exploring the different regions of the country.

Project Briefe:

For this project we were assigned to create a cook book that contains an authentic menu with recipes all around Italy.
Along the way we had some limitations with the color palette and with the pictures we could use. We decided to focus on the seafood recipes giving this book a unique theme.
With these ideas in mind, we came up with the illustrations aiming for the customer to notice what the book main recipes are.
